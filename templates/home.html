Niimport

Niimport enables participants to donate their personal data for academic research with minimal hassle. It connects directly with data portability APIs, which are available under the GDPR right of data access. Niimport allows participants to authorize, review, and revoke their data donation in a few clicks.

For Participants

If you received a donation link from a researcher, use that link to get started.

For Researchers

Use the API to create and manage data donations.
